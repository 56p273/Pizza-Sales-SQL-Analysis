# Pizza Sales SQL Analysis

## 📊 Project Overview
This project focuses on analyzing pizza sales data using SQL queries and Power BI.

The analysis provides business insights related to revenue, sales trends, order patterns, pizza categories, and best-selling pizzas.


## 🛠 Tools Used
- SQL Server Management Studio (SSMS)
- SQL
- Power BI
- Excel


## 📌 SQL Concepts Used
- SELECT Statement
- Aggregate Functions
- GROUP BY
- ORDER BY
- DISTINCT
- CAST
- TOP
- WHERE Clause
- Date Functions

## 📈 Key Business Analysis

### 💰 Revenue Analysis
- Total Revenue
- Average Order Value

### 🍕 Sales Analysis
- Total Pizzas Sold
- Average Pizzas Per Order
- Total Orders

### 📅 Trend Analysis
- Daily Trend for Orders
- Monthly Trend for Orders

### 📊 Category Analysis
- Percentage of Sales by Pizza Category
- Percentage of Sales by Pizza Size

### 🔥 Best & Worst Performing Pizzas
- Top 5 Pizzas by Revenue
- Bottom 5 Pizzas by Revenue
- Top 5 Pizzas by Quantity
- Bottom 5 Pizzas by Quantity
- Top 5 Pizzas by Total Orders
- Bottom 5 Pizzas by Total Orders
